# Doppel-Site

This repository contains the source code for
[Doppel](https://github.com/dr8co/doppel)'s landing page.

Doppel is a CLI tool for finding duplicate files.

It is a simple website built with vanilla HTML, CSS, and JavaScript.

You can view the live site at [https://doppel.draco.codes](https://doppel.draco.codes).
